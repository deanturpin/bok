# Dan prvi - day one
Note *prvi* means *first* not *one*.

* Nedjelja – Sunday: “day with no work”
* Ponedjeljak – Monday: “the day which follows nedjelja“
* Dvadeset pet - 25

See [days of the week](https://www.learncroatian.eu/blog/days-of-week-croatian).
